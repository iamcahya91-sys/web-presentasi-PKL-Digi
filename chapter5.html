Tools yang digunakan saat di CV DIGI Bali


XAMPP



XAMPP adalah perangkat lunak gratis yang berfungsi sebagai server lokal yang berisi Apache, MySQL/MariaDB, PHP, dan Perl, digunakan untuk menjalankan dan menguji website atau aplikasi web secara offline di komputer sebelum diunggah ke server online.



Wordpres.org



WordPress.org adalah platform open-source gratis untuk membuat dan mengelola website yang diinstal secara mandiri di hosting dan domain sendiri, sehingga pengguna memiliki kendali penuh untuk mengatur tema, plugin, dan kode sesuai kebutuhan website.
