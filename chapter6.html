Cara Install Theme di Wordpress




1. Pastikan Folder/file dan tools sudah lengkap (XAMPP, File Wordpress, File Tema.)





2. Nyalakan XAMPP, start Aphace dan MySQL.




3. Buka file Wordpress di local server




4. Setelah login, klik upload theme dan pilih chose file




5. Klik install now, tunggu beberapa saat




6. Install selesai, Pilih Plugin yang ingin digunakan dan klik install dan import demo




7. Setelah memilih tema, klik import full demo




8. Finish, artinya install tema sudah selesai




9. Klik visit site, akan muncul tampilan website
